Дженерики - это параметризированные типы.
Дженерики позволяют объявлять классы, интерфейсы и методы, где тип данных, которыми они оперируют,
указан в виде параметра.

Дженерики работают только с объектами. Нельзя параметризовать класс, интерфейс или метод примитивами.
Параметризация на уровне класса обозначается символом <T> после имени класса.
Наследники базового класса аналогично должны быть обозначены данной параметризацией <T>.
Пример создания объекта параметризированного класса: Archer<Bow> archer = new Archer<>.
Параметризация на уровне метода обозначается символом <T> перед возвращаемым типом метода.
Параметризация на уровне метода может быть обозначена при помощи символа wildcard (<?>).
Параметризация на уровне метода может быть ограничена "сверху" через запись <? extends...> и "снизу" через запись
<? super ...>.

Extends используется для producers. Super используется для consumers
